NBA Fan Hub – Site Plan

Site Name

NBA Fan Hub – This name was chosen because it reflects a central, interactive destination for fans to explore teams, players, and NBA updates. It is simple, recognizable, and directly tied to basketball culture.

Site Purpose

The purpose of this website is to provide NBA fans with an interactive hub featuring team information, player stats, seasonal highlights, and the ability for users to save favorite players using localStorage. It will also include a contact form and a references page.

Scenarios

  • "Which NBA players are performing the best this season?"
  • "How can I quickly find information about my favorite team?"

Color Schema

The website will use the following colors:

  • NBA Red (#C9082A) – Used for accents, buttons, and interactive elements.
  • NBA Blue (#17408B) – Used for headings and navigation.
  • Light Gray (#F2F2F2) – Used for background areas to improve readability.

The colors visible throughout this page reflect the chosen color scheme.

Typography

Roboto will be used as the primary font throughout the website.

  • Headings – Roboto Bold
  • Body Text – Roboto Regular

This page uses Roboto to demonstrate the typography plan.
